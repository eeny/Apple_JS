서버 : 접속자가 요청을 하면 데이터를 갖다줌
        요청 : GET / POST
        GET : URL을 입력해서 요청함 (읽기)
        POST : 숨겨서 뭔가 정보를 전달하거나 요청함 (쓰기)
        
        AJAX : 서버에다가 요청을 할 수 있게 도와주는데
        새로고침 없이 요청할 수 있음
        
        예제 임시서버
        https://codingapple1.github.io/hello.txt
        https://codingapple1.github.io/data.json
